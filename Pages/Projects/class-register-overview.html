E-Napló

Egy ASP.NET MVC alapú E-Napló. A projektből még hiányzik pár alapvető funkció (jelen állapotában az admin funkciók vannak készen).
Az ötlet egy iskolai napló webapp, inkább egy általános, középiskola naplójához hasonlító pl. Kréta.
Az adatbázist Entity Framework code first approach-al készült.
